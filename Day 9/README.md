# Flutter Animations & Transitions App

A Flutter application demonstrating various animation techniques including implicit animations, explicit animations, hero transitions, and custom page transitions. This project helps developers understand Flutter animation APIs and create smooth, interactive user experiences.

---

## 📌 Features

### ✨ Implicit Animations
- AnimatedContainer
- AnimatedOpacity
- AnimatedPositioned
- AnimatedCrossFade
- TweenAnimationBuilder

### 🎬 Explicit Animations
- AnimationController
- Tween Animations
- Animation Listeners
- Proper Controller Disposal

### 🚀 Hero Animations
- Hero Widget Transitions
- Tag Matching Between Screens
- Smooth Shared Element Animations

### 🔄 Custom Transitions
- Fade Transitions
- Slide Transitions
- Reusable Animation Builders

---

## 🎯 Learning Objectives

- Understand Flutter animations
- Create implicit animations
- Build explicit animations
- Implement custom transitions
- Learn animation curves and timing
- Develop animated and interactive UIs
