## Animations Catalog (this project)

This workspace contains an interactive catalog of Flutter animation examples built to match a learning plan:

- Implicit animations: `AnimatedContainer`, `AnimatedOpacity`, `AnimatedCrossFade`, `TweenAnimationBuilder`
- Explicit animations: `AnimationController`, `Tween`, `ScaleTransition`
- Hero animations and custom transitions with `flightShuttleBuilder` and `MaterialRectArcTween`
- `AnimatedBuilder` for optimized rebuilds
- Small animated UI examples (login form reveal, loading state)
